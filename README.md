# prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Create account
- [ ] Request Secret
- [ ] Confirm Secret (Login)
- [ ] Like /UnLike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] See my profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
- [ ] Send private Message
- [ ] See rooms
- [ ] See room
- [ ] Receive Message (Realtime)
